STOCK PREDICTION MODEL

Published in International Journal for Innovative Technology and Exploring Engineering, June 2020.


The Stock Market is a challenging forum for investment. This project aims at processing large volumes of data and running comprehensive regression algorithms on the dataset; that will predict the future value of a stock. MONTAGE WIZARD

Cloud-enabled montage generator using Twitch API and EasyOCR.


Twitch is one of the leading live-streaming platforms for E-Sports today. In a week around 70-80 hours of gameplay per streamer is streamed and uploaded on Twitch. Since the average attention span of a viewer is not more than a couple minutes, the streamer needs to spend hours on end editing and clipping his videos. These clips are called Montages and include game highlights like high intensity battles, and clutch plays. Using Twitch API to source videos and EasyOCR to identify target montages, GAE-enabled Montage Wizard helps to automate this process, thereby conserving precious time and resources.

AWS IMAGE CLASSIFIER

ML based Image Classification system deployed on AWS[IAAS].


In this project, we aim to deploy a Cloud based model which has an underlying Image Classification Algorithm. The ML based image classifier processes thousands of images and returns the result as a tuple (Filename, Classification), displaying results as they are calculated by the different EC2 instances. Depending on the traffic of incoming user requests, the system is able to autoscale the number of app tier instances. This gives us hands-on experience on AWS EC2, SQS and S3.

PYTORCH VS. APACHE SPARK

A comparative study between PyTorch and Apache Spark.


PyTorch has become widely used over the last few years, and studies have been performed to compare it to other frameworks like Tensorflow. However there have been relatively few analyses made between the general use of PyTorch and the use of a large, distributed framework such as Hadoop or Spark. Therefore, in this project, we have taken this opportunity to try our hand at comparing Image Classification accuracies drawn by two models: PyTorch and Apache Spark.
